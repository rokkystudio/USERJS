# USERJS

Набор браузерных интеграций для сайтов через расширение **User JavaScript and CSS**.

В репозитории собраны отдельные JS- и CSS-проекты, которые подключаются в браузере через пользовательские правила расширения.

## Что здесь находится

- JS-интеграции для клиентских сценариев
- CSS-кастомизации интерфейсов
- отдельные проекты по папкам

## Как использовать

1. Установить расширение **User JavaScript and CSS** из Chrome Web Store.
2. Открыть нужный проект в репозитории.
3. Скопировать JS- и/или CSS-код в правило расширения.
4. Привязать правило к нужному сайту или URL-маске.

## Расширение

- Chrome Web Store: [User JavaScript and CSS](https://chromewebstore.google.com/detail/user-javascript-and-css/nbhcbdghjpllgmfilhnhkllmkecfmpld)
