# GPT Code Colors

Небольшой проект для переопределения оформления кода в ChatGPT.

## Зачем

Изначально оформление кода в интерфейсе ChatGPT было комфортным для чтения, но позднее визуальный стиль изменился и стал заметно агрессивнее для глаз при длительной работе. Этот проект возвращает более спокойное восприятие кода и ориентируется на визуальную подачу, близкую к IDE от JetBrains.

## Что делает проект

Проект переопределяет внутренние классы оформления редактора и кодовых блоков в интерфейсе ChatGPT. Основная задача — изменить отображение синтаксиса, базового текста и связанных элементов так, чтобы код читался мягче и стабильнее при длительной сессии.

## Как это устроено

- проект работает через CSS-переопределения;
- использует внутренние классы, которые ChatGPT назначает блоку кода и токенам подсветки;
- может дополняться новыми правилами по мере появления новых элементов интерфейса и новых токенов.

## Особенности

В проекте не фиксируется окончательный набор правил. ChatGPT периодически меняет внутреннюю разметку и служебные имена классов, поэтому таблица переопределений может расти вместе с развитием интерфейса.

## Назначение

Проект подходит для тех, кто:

- много читает код прямо в ChatGPT;
- хочет снизить визуальную перегрузку;
- предпочитает более привычную IDE-подачу синтаксиса.

## Дальнейшее развитие

В репозиторий можно добавлять новые переопределения для:

- дополнительных токенов синтаксиса;
- состояний выделения и фокуса;
- встроенных редакторов и блоков предпросмотра;
- изменений внутренней разметки после обновлений ChatGPT.
