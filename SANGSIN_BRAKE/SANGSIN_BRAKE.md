# SANGSIN ARTICLE COPY FIX

Этот скрипт делает доступным для выделения и копирования артикул запчасти на сайте e-sangsin.com.

Подходит для продавцов запчастей, которым нужно быстро забирать номер детали из каталога без ручного переписывания.

После подключения:
- артикул можно выделить и скопировать;
- клик по картинке и `more` по-прежнему открывает каталог;
- работает в Chrome и Firefox.
