# WIKI SAMARITAN

WIKI SAMARITAN — это CSS-скрипт для корректного отображения самаритянского письма на страницах вики.

## Зачем нужен

На некоторых страницах Wikipedia самаритянские символы отображаются некорректно: браузер использует шрифт сайта, который не поддерживает это письмо, из-за чего вместо текста появляются квадраты.

Этот CSS принудительно задаёт шрифт с поддержкой самаритянского письма и позволяет отображать текст читаемо.

## Что делает

- назначает `Noto Sans Samaritan` основным шрифтом для нужных элементов;
- добавляет запасные Unicode-шрифты;
- исправляет отображение самаритянских символов в содержимом страницы.

## Назначение

Проект подходит для пользовательских стилей, локальных CSS-правил и вики-страниц, где требуется стабильное отображение самаритянского письма.
